FORTUS™ Version 5.8 Build 1

Automobile

  • Added many new fields to LeadInfo, in the Extended Data Model: ContactLanguage, ContactMethod, ContactTime, LeadTypeEn, LeadTypeFr, RecipientAccountCode, RecipientAccountName, Referrer, and SubmitterLeadID. See LeadDetail for more details.
  • Added new messages: 315. See Auto Messages for more details.

Habitation

    Hab data model

    • THIS IS A BREAKING CHANGE!
      Added new extended coverage aliases: OverlandWaterExt. This new single alias REPLACES all 3 previous aliases. The following are NO LONGER VALID and SHOULD NOT BE USED: OverlandWaterBldgExt, OverlandWaterCntsExt, OverlandWaterOutBldgExt. See Structure Aliases for more details.
    • Added new messages: 72 and 77. See Hab Messages for more details.
    • Removed messages: 331, 332, and 339.

    Hab2 data model

    • THIS IS A BREAKING CHANGE!
      Added new extended coverage type: OverlandWater. This new single type REPLACES all 3 previous types. The following are NO LONGER VALID and SHOULD NOT BE USED: OverlandWaterBldg, OverlandWaterCnts, OverlandWaterOutBldg. See Table 42 for more details.
    • Added new messages: 72 and 77. See Hab Messages for more details.
    • Removed messages: 331, 332, and 339.
    • Added many new fields to LeadInfo, in the Extended Data Model: ContactLanguage, ContactMethod, ContactTime, LeadTypeEn, LeadTypeFr, RecipientAccountCode, RecipientAccountName, Referrer, and SubmitterLeadID. See LeadDetail for more details.
