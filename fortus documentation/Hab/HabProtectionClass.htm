Protection Classes

There are 3 protection classes, see Table 1. However, some carriers break down the Semi-Protected class further based on the distance to the nearest firehall. FORTUS™ thus subclasses Semi-Protected, see Table 2. Table 2 values are interal to FORTUS™, they are NOT valid <ProtectionClass> values. Table 3 describes all the possible permutations of input values for the <ProtectionClass>, <FHDist>, and <HydDist> tags.

Table 1: <ProtectionClass> Values
Range Description
P Protected
S Semi-Protected
U Unprotected
top

Table 2: Internal Semi-Protection SubClasses
Range Description
S1 Semi-Protected 1
S2 Semi-Protected 2
S3 Semi-Protected 3

Examples

Example 1
This example will be rated as Protected because the ProtectionClass is 'P' in the input. The FHDist and HydDist values are irrelevant.
Input:
   ProtectionClass = 'P'
   FHDist = 40
   HydDist = 600
top

Example 2
This example will be rated as Unprotected because the ProtectionClass is 'U' in the input. The FHDist and HydDist values are irrelevant.
Input:
   ProtectionClass = 'U'
   FHDist = 3
   HydDist = 10
top

Example 3
This example will be rated as Protected because:
   ProtectionClass is not 'P' or 'U'
   FHDist <= 10 (Protected threshold for carrier in this example)
   HydDist <= 300
Assumptions:
   We will assume that the carrier being quoted defines Protected as being within 10 kms of a firehall.
Input:
   ProtectionClass = ''
   FHDist = 10
   HydDist = 150
top

Example 4
This example will be rated as Semi-Protected (with Semi-Protected 2 subclass being used internally) because:
   ProtectionClass is not 'P' or 'U'
   FHDist > 8 (Semi-Protected 1)
   FHDist <= 12 (Semi-Protected 2)
Assumptions:
   We will assume that the carrier being quoted has three possible Semi-Protected ratings. Anything within 8km of a firehall is Semi-Protected 1, within 13km is Semi-Protected 2, and within 20km is Semi-Protected 3.
Input:
   ProtectionClass = ''
   FHDist = 10
   HydDist = 1500
