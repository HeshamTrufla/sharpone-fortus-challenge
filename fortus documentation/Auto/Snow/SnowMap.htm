Overview

Rating Snow Vehicles is built on top of the normal Automobile hierarchy. Therefore, to construct quote input for Snow Vehicles, first begin by referring to the normal Automobile Map Definition and constructing input based on this information. However, there are a few pieces of data that are different for Snow Vehicles. These details have been listed below.
