VICC Information

Each carrier defines their own set of VICC Vehicle Rate Groups. However, before calculation of the quote, a facility is needed to determine if a particular vehicle year and vehicle model are valid (Ex. Did the Triumph Street Triple exist in 2014?). A file, MOTODETL_BI.TXT, is supplied to fill this need.

The first line (or record) contains the field names. This line should be deleted before or after the data is imported into the database. Each field is delimited by a pipe ('|') character, and each line (or record) is seperated by a carriage return and a line feed. The fields are defined in Table 1 below.

Table 1: MOTODETL_BI.TXT Fields
Field Type Width Description
MODEL_YEAR NUMBER 4 Vehicle model year. MODEL_YEAR and EXT_VICC make a primary key for this data.
EXT_VICC CHAR 6 Extended VICC vehicle code. MODEL_YEAR and EXT_VICC make a primary key for this data.
VICC_CODE CHAR 4 VICC vehicle code
MAKE CHAR 25 Vehicle make, a.k.a. vehicle manufacturer
MAKE_FR CHAR 25 French version of MAKE.
MODEL CHAR 40 Vehicle model
MODEL_FR CHAR 40 French version of MODEL.
MAKEMOD CHAR 20 Abbreviation of MAKE and MODEL
MODEL_FR CHAR 20 French version of MAKEMOD.
ABS CHAR 1 Anti-lock bracking system.
  • 0 - not available
  • 3 - optional
  • 4 - standard
WEIGHT CHAR 4 Measured in kilograms (kg). Blank, or a value of 0, indicates not available.
PRICE CHAR 6 Manufacturer's Suggested Retail Price + GST. Blank, or a value of 0, indicates not available.
CYLINDER CHAR 2 Engine cylinder.
  • 1, 2, 3, 4, 6 and 8 = number of cylinders
  • 0 - rotary engine, no cylinder
  • 60 - Electric motor, no cylinder
  • 99 - information not known
FUELTYPE CHAR 1 Engine fuel type.
  • G - Gasoline
  • D - Diesel
  • V - Electric, no fuel
  • N - Natural gas
  • X - Not available or Unknown
BODYSTYLE CHAR 2 Body style (older classification method)
  • 3M - 3 Wheel motorcycle
  • 3S - 3 Wheel scooter
  • C - custom
  • CS - custom sidecar
  • DP - dual purpose
  • MA - adventure
  • MP - moped/scooter
  • OE - off-road enduro
  • OM - off-road motocross
  • OR - off-road
  • OT - off-road trial
  • S - sport
  • T - touring
  • TR - 3 Wheel motorcycle 'trike'
  • V - vintage
  • XX - unknown
BODYSTYLE2 CHAR 2 Body style 2 (newer classification method)
  • SA - Standard
  • CR - Cruiser
  • NS - Naked Sport
  • S - Sport
  • SG - Sport Touring
  • T - Touring
  • SS - Standard Sidecar
  • MP - Moped
  • SR - Scooter
  • DP - Dual Purpose
  • MA - Adventure
  • OE - Enduro
  • OS - Motard / Super Motard
  • OR - Off Road
  • OM - Off Road Motocross
  • OT - Off Road Trials
  • 3S - 3 Wheel Scooter (Excludes 2 rear wheels)
  • 3M - 3 Wheel Motorcycle (Excludes 2 rear wheels)
  • TR - Trike (3 Wheel Motorcycle)
  • V - Vintage
  • XX - N/A or Unknown class type
BODYTYPE CHAR 2 CSIO body type. Blank at the moment. For future use.
DRIVETRAIN CHAR 2 Drive train
  • B - belt drive
  • C - chain drive
  • S - shaft drive
  • X - unknown
VEHTYPE CHAR 2 Vehicle type
  • MC - motorcycle
CC CHAR 4 Engine Displacement (cc)
  • 0 - electric motor
COOLING CHAR 2 Engine Cooling. Blank indicates electric motor.
  • L - Liquid
  • A - Air
  • AO - Air/Oil

  • F - Fan

ENG_CONFIG CHAR 3 Engine Configuration
  • F1 = Flat - 1 cylinder
  • H2 = Horizontally Opposed - 2 cylinders
  • H4 = Horizontally Opposed - 4 cylinders
  • H6 = Horizontally Opposed - 6 cylinders

ENG_STROKE CHAR 1 Engine Stroke
  • 0 = electric motor

Frequently Asked Questions

Q: Do I need all these fields, all this data?
A: No, not all of this data is necessary. A lot of the data may never be used.
Q: If most of this data is not necessary, why is it there?
A: Some of the data may be used in other ways, in very different applications. Some data may be used in the future. Etc.
Q: Some of the entries seem to be missing some data. Why is there missing data?
A: The model year, VICC codes and make/model fields should always be populated. After that, it is possible that description details about the vehicle are not present. Systems using this description data must plan for this scenario.
Q: We are currently using MCVICC_BI.TXT. Do we need to switch to MOTODETL_BI.TXT?
A: Yes, you should make plans to change to MOTODETL_BI.TXT (MCVICC_BI.TXT will continue to remain available for some time). There are several benefits with the new data, but having a list that goes beyond the last 15 years is a major reason.
