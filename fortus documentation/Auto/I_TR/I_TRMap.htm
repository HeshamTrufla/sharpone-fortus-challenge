Overview

Rating Commercial Trailers is built on top of the normal Automobile hierarchy. Therefore, to construct quote input for Commercial Trailers, first begin by referring to the normal Automobile Map Definition and constructing input based on this information. However, there are a few pieces of data that are different for Commercial Trailers. These details have been listed below.

QuoteInfo
Name Type Length Range/Format Required Description
CommAssumeMultiLine BOOL   Y/N   Works the same way as AssumeMultiLine, except that it works with CommHabCompanyCode. Default: 'N'. See Data Propagation for more information.
CommHabCompanyCode STRING 6     The CSIO code for the company with whom commercial property insurance is held.
CommHabDateCompany DATE   YYYYMMDD   The inception date of the insured's commercial property insurance policy with the company specified in ComHabCompanyCode.
