VICC Information

Each carrier defines their own set of VICC Vehicle Rate Groups. However, before calculation of the quote, a facility is needed to determine if a particular vehicle year and vehicle model are valid (Ex. Did the Ford Tempo exist in 1992?). A file, PPVDETL_BI.TXT, is supplied to fill this need.

The first line (or record) contains the field names. This line should be deleted before or after the data is imported into the database. Each field is delimited by a pipe ('|') character, and each line (or record) is seperated by a carriage return and a line feed. The fields are defined in Table 1 below.

Table 1: PPVDETL_BI.TXT Fields
Field Type Width Description
MODEL_YEAR NUMBER 4 Vehicle model year. MODEL_YEAR and EXT_VICC make a primary key for this data.
EXT_VICC CHAR 6 Extended VICC vehicle code. MODEL_YEAR and EXT_VICC make a primary key for this data.
VICC_CODE CHAR 4 VICC vehicle code
MAKE CHAR 20 Vehicle make, a.k.a. vehicle manufacturer
MAKE_FR CHAR 20 French version of MAKE.
MODEL CHAR 40 Vehicle model
MODEL_FR CHAR 40 French version of MODEL.
MAKEMOD CHAR 20 Abbreviation of MAKE and MODEL
MODEL_FR CHAR 20 French version of MAKEMOD.
WHEELBASE CHAR 4 Measured in millimeters (mm). Blank, or a value of 0, indicates not available.
WEIGHT CHAR 4 Measured in kilograms (kg). Blank, or a value of 0, indicates not available.
HORSEPOWER CHAR 3 Engine horsepower. Blank, or a value of 0, indicates not available.
PRICE CHAR 6 Manufacturer's Suggested Retail Price + GST. Blank, or a value of 0, indicates not available.
AIRBAGS CHAR 1 Vehicle airbags.
  • 0 - not available
  • 1 - optional
  • 2 - standard on driver's side
  • 3 - standard on driver's side / optional on passenger's side
  • 4 - standard dual airbags
  • 5 - standard dual and additional airbags (e.g. side airbags)
  • 6 - standard dual and optional additional airbags (e.g. side airbags)
ABS CHAR 1 Anti-lock bracking system.
  • 0 - not available
  • 1 - rear wheels ABS only
  • 2 - rear wheels only / four wheels ABS optional
  • 3 - optional
  • 4 - standard
ALM_AUD CHAR 1 Theft-deterent system - audible alarm.
  • 0 - not available
  • 1 - standard alarm
  • 2 - optional alarm
ALM_CUTOFF CHAR 1 Theft-deterent system - fuel / ignition / electrical cut-off system.
  • 0 - not available
  • 1 - standard system
  • 2 - optional system
ALM_SECKEY CHAR 1 Theft-deterent system - security key system.
  • 0 - not available
  • 1 - standard system
  • 2 - optional system
CYLINDER CHAR 2 Engine cylinder.
  • 2, 3, 4, 5, 6, 8, 10 and 12 = number of cylinders
  • 0 - rotary engine, no cylinder
  • 50 - Electric motor, no cylinder - Low Speed Vehicle (LSV)
  • 54 - Electric motor with 4 cylinder engine for extended range
  • 60 - Electric motor, no cylinder
  • 99 - information not known
FUELTYPE CHAR 1 Engine fuel type.
  • G - Gasoline
  • D - Diesel
  • V - Electric, no fuel
  • N - Natural gas
  • X - Not available or Unknown
INDUCTION CHAR 1 Engine - forced induction
  • T - Turbo
  • S - Supercharged
HYBRID CHAR 1 Engine - hybrid
  • H - Hybrid
TRACCTRL CHAR 1 Traction control
  • 0 - Not available
  • 1 - Standard
  • 2 - Optional
STABCTRL CHAR 1 Stability control
  • 0 - Not available
  • 1 - Standard
  • 2 - Optional
BODYSTYLE CHAR 2 Body style
  • 2D - 2 Door (car)
  • 4D - 4 Door (car)
  • 1P - ½ Ton Pickup
  • 2P - ¾ Ton Pickup
  • 3P - 1 Ton Pickup
  • 4P - Compact Pickup
  • CV - Cargo Van
  • MP - Multi-Purpose (SUV)
  • PV - Passenger Van
  • SS - Sport / Specialty
  • SW - Station Wagon
BODYTYPE CHAR 2 CSIO body type
  • JP - small SUV (convertible)
  • MV - minivan
  • P1 - PPV - convertible
  • P2 - PPV - 2 door coupe/sedan
  • P3 - PPV - 2 door hatchback
  • P4 - PPV - 4 door sedan
  • P5 - PPV - 4 door hatchback
  • P6 - station wagon
  • PV - pickup truck
  • ST - small soft top SUV
  • SU - small hard top - medium SUV
  • SW - large SUV
  • VL - full size van
DRIVETRAIN CHAR 2 Drive train
  • 1 - 2 wheel drive
  • 2 - 4 wheel drive
  • 1F - front wheel drive (not yet available)
  • 1R - rear wheel drive (not yet available)
VEHTYPE CHAR 2 Vehicle type
  • PP - private passenger
  • TR - truck
CNVERTIBLE CHAR 1 Convertible
  • 1 - convertible
  • 0 - not convertible

Frequently Asked Questions

Q: Do I need all these fields, all this data?
A: No, not all of this data is necessary. A lot of the data may never be used.
Q: If most of this data is not necessary, why is it there?
A: Some of the data may be used in other ways, in very different applications. Some data may be used in the future. Etc.
Q: Some of the entries seem to be missing some data. Why is there missing data?
A: The model year, VICC codes and make/model fields should always be populated. After that, it is possible that description details about the vehicle are not present. Systems using this description data must plan for this scenario.
Q: We are currently using VICC_BI.TXT. Do we need to switch to PPVDETL_BI.TXT?
A: Yes, you should make plans to change to PPVDETL_BI.TXT (VICC_BI.TXT will continue to remain available for some time). There are several benefits with the new data, but the main force driving the change will be the need to use the 6-digit Extended VICC code, instead of just the 4-digit VICC code. Increasingly, companies need the 6-digit extended code to quote accurately.
