Commercial VICC Information

Each carrier defines their own set of VICC Vehicle Rate Groups.   However, before calculation of the quote, there is a need to guide the user to the selection of the proper vehicle and collect known details about the vehicle to help ensure accurate rating. For example, did the Mazda MPV 2WD exist in 2001, and if so, what was the size of the engine?  The following files are provided to provide this functionality:

  • ICVICC_BI.TXT
  • ICBASEYEAR.DAT
  • ICDESCRIPT_BI.TXT

ICVICC_BI.TXT

This file can be used to validate up-front what vehicles existed in what years. NOTE that this file is only accurate for the last 14 years; 15 years and older are grouped into one column. The first line (or record) contains the field names. The second line is an empty line/record. These two lines should be deleted before or after the data is imported into the database. Each field is delimited by a pipe ('|') character, and each line (or record) is seperated by a carriage return and a line feed. The fields are defined in Table 1 below.

Table 1: ICVICC_BI.TXT Fields
Field Type Width Description
EXT_VICC_CODE CHAR 6 Extended VICC Code
VICC_COD CHAR 4 VICC Code
MAKE_AND_M CHAR 35 Make and model Name
MANUFAC CHAR 20 Manufacturer Name
SHORT_FO CHAR 20 Short form of MANUFAC and MAKE_AND_M
YEAR_15 CHAR 2 BASEYEAR
YEAR_14 CHAR 2 BASEYEAR - 1
YEAR_13 CHAR 2 BASEYEAR - 2
YEAR_12 CHAR 2 BASEYEAR - 3
YEAR_11 CHAR 2 BASEYEAR - 4
YEAR_10 CHAR 2 BASEYEAR - 5
YEAR_9 CHAR 2 BASEYEAR - 6
YEAR_8 CHAR 2 BASEYEAR - 7
YEAR_7 CHAR 2 BASEYEAR - 8
YEAR_6 CHAR 2 BASEYEAR - 9
YEAR_5 CHAR 2 BASEYEAR - 10
YEAR_4 CHAR 2 BASEYEAR - 11
YEAR_3 CHAR 2 BASEYEAR - 12
YEAR_2 CHAR 2 BASEYEAR - 13
YEAR_1 CHAR 2 BASEYEAR - 14
YEAR_0 CHAR 2 (BASEYEAR - 15) or prior
DELFLAG CHAR 1 Ignore
top

  • YEAR_?? can contain any of the following:
    • 1 or 2 digit number, or 'A'.  This means that this Manufac/Make combination existed for this Year.
    • ' ','-', or other.  This means that this Manufac/Make combination did not exist for this Year.
  • For model years dated after BASEYEAR use BASEYEAR.
  • For model years dated before BASEYEAR-15 use BASEYEAR-15

ICBASEYEAR.DAT

ICBASEYEAR.DAT contains only 4 characters.  These 4 characters form the BASEYEAR described in Table 1.

ICDESCRIPT_BI.TXT

This file should be used to populate additional information about the vehicle. The first line (or record) contains the field names. This line should be deleted before or after the data is imported into the database. Each field is delimited by a pipe ('|') character, and each line (or record) is seperated by a carriage return and a line feed. The fields are defined in Table 2 below.

Table 2: ICDESCRIPT_BI.TXT Fields
Field Type Width Description Corresponding Map Field
VICC_CODE CHAR 4 VICC Code <Vehicle><VICCCode>
EXT_VICC_CODE CHAR 6 Extended VICC Code <Vehicle><VICCCodeExt>
MODEL_YEAR LONG Year <Vehicle><Year>
WHEELBASE CHAR 4 Wheelbase (mm) Not used.
WEIGHT CHAR 6 Gross Vehicle Weight (kg) <Vehicle><GVW>
HORSEPOWER CHAR 3 Horsepower Not used.
PRICE CHAR 6 Value of vehicle when it was new, i.e. MSRP <Vehicle><ValueNew>
AIRBAGS CHAR 1 Airbags Not used.
ABS CHAR 1 ABS Not used.
ALM_AUDIBLE CHAR 1 Theft-Deterent Device - Audible Alarm Not used.
ALM_CUTOFF CHAR 1 Theft-Deterent Device - Fuel/Ignition/Electrical Cut-off System Not used.
ALM_SECURITYKEY CHAR 1 Theft-Deterent Device - Security Key System Not used.
ENGINE_CYL CHAR 2 Number of engine cylinders Not used.
ENGINE_FUEL CHAR 1 Engine fuel type <Vehicle><FuelType>
ENGINE_INDUCT CHAR 1 Engine - Forced Induction Not used.
ENGINE_HYBRID CHAR 1 Engine - Hybrid Not used.
TRACTION_CTRL CHAR 1 Traction Control Not used.
STABILITY_CTRL CHAR 1 Stability Control Not used.
BODY_STYLE CHAR 2 Body Class/Style Not used.
DRIVE_TRAIN CHAR 3 Drive Train Not used.
